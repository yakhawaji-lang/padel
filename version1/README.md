# Padel Tournament - Version 1

This is Version 1 of the Padel Tournament application.

## Features

- **King of the Court Mode**: 8 teams, each team plays 7 matches
- **Social Tournament Mode**: 12 teams divided into 4 groups of 3, with semi-finals and finals
- **Bilingual Support**: English and Arabic with RTL support
- **Match Timer**: 15-minute timer with pause/reset functionality
- **Standings Tables**: Individual group standings for Social Tournament, overall standings for King of Court
- **Match History**: Collapsible section showing all match results
- **Match Schedule**: Collapsible section for King of Court showing team opponents (played and remaining)
- **Draw Support**: Teams can tie matches
- **Drag and Drop**: Manual team assignment for Social Tournament groups

## To Restore This Version

When you want to restore this version, say: **"put the work back to version 1"**

The assistant will copy all files from the `version1` folder back to the main `src` folder.
