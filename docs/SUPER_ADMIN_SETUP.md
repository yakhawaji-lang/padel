# إعداد Super Admin (مدير النظام)

## بعد إعادة تهيئة قاعدة البيانات

عند تنفيذ إعادة تهيئة كاملة (`/api/init-db?reset=1`):

- تُحذف جميع البيانات
- يُنشأ مستخدم Super Admin واحد فقط
- لا تُنشأ أي نوادي افتراضية

## بيانات الدخول الافتراضية

| الحقل | القيمة |
|-------|--------|
| **البريد الإلكتروني** | `admin@playtix.app` |
| **كلمة المرور** | `Admin@123456` |

## رابط الدخول

- **رابط مستقل:** `/super-admin` أو `https://your-domain.com/super-admin`
- يمكن أيضاً استخدام `/admin-login` (يوجّه لنفس صفحة الدخول)

## صلاحيات Super Admin

- التجول في النظام كاملاً
- إضافة مستخدمين للنظام (Admin Users)
- إضافة أندية (Manage All Clubs)
- إضافة أعضاء (All Members)
- التحكم الكامل بجميع الإعدادات

## تغيير كلمة المرور

يُنصح بتغيير كلمة المرور الافتراضية فوراً بعد أول دخول عبر:
- صفحة إدارة المستخدمين (Admin Users)
- أو تنفيذ استعلام SQL لتحديث الحقل `password_hash` في جدول `platform_admins`
