# صور النوادي | Club Images

## إنشاء مجلد لنادي جديد

1. انسخ مجلد `_template` وسمّه `[club-id]` (مثل `hala-padel` أو `best-padel-club`)
2. أضف الصور في المجلدات:
   - **logo/** — شعار النادي (مربع أو صورة شفافة)
   - **banner/** — بنر عرضي للصفحة العامة
   - **courts/** — صور الملاعب (court-1.png، court-2.png، ...)
   - **offers/** — صور العروض الترويجية

## أسماء الملفات المقترحة

| النوع | المثال |
|------|--------|
| الشعار | `logo.png` |
| البنر | `banner.png` |
| الملاعب | `court-1.png`, `court-2.png` |
| العروض | `offer-1.png`, `offer-summer.png` |

## الربط مع التطبيق

صور النوادي تُخزّن في قاعدة البيانات. يمكن:
- رفع الصور من لوحة تحكم النادي (Club Settings)
- أو استخدام روابط من خادم خارجي
- أو نسخ الصور من هذا المجلد عند النشر
